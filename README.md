⚡Campus Energy Consumption Forecasting System

A Machine Learning & Time-Series based web application that predicts future campus energy consumption using historical electricity usage data.

📌 Project Overview

This project is a Flask-based web application that forecasts energy usage using:

📊 Linear Regression (Machine Learning)

📈 ARIMA (Time-Series Forecasting)

📁 CSV Dataset Input

🌐 Flask Web Framework

It helps institutions analyze and predict electricity consumption for smarter planning and sustainability.

🎯 Problem Statement

Campuses consume large amounts of electricity daily. Without forecasting:

Energy waste increases

Operational costs rise

Load management becomes difficult

This system provides data-driven predictions to support:

Cost optimization

Sustainability planning

Smart campus initiatives

Energy demand forecasting
